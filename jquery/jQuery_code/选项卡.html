锦绣河山，江涛滚滚，鸿越鹰翔。看画桥飞渡，霞楼耸秀；绮船穿越，云舞盛阳。 翠柏青松，习风飒爽，猎猎旌旗万里扬。慎思量，任凭栏诗酒，漫道沧桑。 前贤夙志皇皇，后承继，共图国富强。笑河虾挥剪，海鳅跳浪；窥仓地鼠，吠日天狼。 可晓天高，可知地厚，龙跃苍穹一扫光。轩辕慰，致大同世界，从此安康。
