生活
过好当下的生活，享受人生的快乐，人生到世来一回，东奔西跑去挣钱，为什么总是心高，为什么越活越累。酸甜苦辣尝个遍，不管到那说不算。吃苦受累没人痛，没有文化就不行。翻山越岭去爬岥，干的都是体力活， c，老板立将你刷了，想要生存多挣钱，走到那里別讨贤，没有本事別强，诚实做事最理想。人生就该这样活，神仙拿你都没辙。快快乐乐度终生，没留遗憾最轻松。人活一世不轻松，累死累活争西东。不吃不喝攒钱财，闭眼以后两手空。留下金山整一座，只换儿女哭几声，活着他们不行孝，死后大办呈英雄，风风雨雨一辈子，坎坎坷坷过一生，大起大落经历过，大悲大喜感受过，就算万般的不易，也要自己去面对，千般难自己打住，万般累自己挺住。
感悟

无奈相思
无奈相思，一卷红尘，是谁的挂牵，人生的无缘，一个错过，一个生死不见，爱梦断，人回首，只是江湖冷三生，岁月无情。一杯老酒，一段人生，沧海无奈，思念自己的孤独，伤害最后的生命。一段思念，一段无缘，守望人海的冷，憔悴自己的辜负。
我的前半生
1、没有任何人会成为你以为的、今生今世的避风港，只有你自己，才是自己最后的庇护所。再破败再简陋，也好过寄人篱下。
　　 　　2、首先要做到可以取代任何人，然后再考虑做到任何人都不可以取代你。
　　 　　3、路要自己一步步走，苦要自己一口口吃，抽筋扒皮才能脱胎换骨。
　　 　　4、人与人之间，在同一时间同一件事情上，只能保留一种关系，否则就会感情用事。
　　 　　5、有来有往是好事，单方面的付出叫什么，叫牺牲，一个人有多少东西可以牺牲，牺牲掉了，就完了，就没有了。
　　 　　6、菩萨心肠，金刚手段，才是对待朋友和你在意的人应该有的态度。
　　 　　7、最佳的报复不是仇恨，而是打心底发出的冷淡，干嘛花力气去恨一个不相干的人。
　　 　　8、金钱与爱情面前卖弄自尊，是最愚蠢的事。
　　 　　9、冷静，不发泄不指责，他就永远欠你的。
